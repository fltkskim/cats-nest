## Backend (port: 8000)
```shell
$ cd backend
$ npm i
$ npm run start
```

## Frontend (port: 3000)
```shell
$ cd frontend
$ npm i
$ npm run dev
```

## backend dotenv
* MONGODB_URI
* PORT=8000
* MODE='dev'
* JWT_SECRET
* SWAGGER_USER
* SWAGGER_PASSWORD# cats-nest
